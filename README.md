# IntroducaoProgramacao_FURB_2026_1
# ☕ Fundamentos de Programação em Java: Estruturas Sequenciais

Este repositório contém uma série de algoritmos desenvolvidos em **Java** com foco na resolução de problemas práticos matemáticos e de negócios. Ele documenta meus passos iniciais na Ciência da Computação, traduzindo o raciocínio analítico para a lógica de programação.

Cada arquivo foi construído seguindo uma metodologia de análise prévia (definição de Entradas, Processamento, Saídas e Fluxograma) antes da implementação do código.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Bibliotecas:** `java.util.Scanner` (para manipulação de entradas via terminal)
* **Conceitos aplicados:** Variáveis (int, float, double), operadores aritméticos, fluxo sequencial de execução e formatação de saída.

## 📁 Estrutura dos Exercícios

Aqui está um resumo das soluções desenvolvidas:

* **[Uni3Exe01](Uni3Exe01.java) - Cálculo de Área:** Algoritmo que lê as dimensões de um terreno e calcula sua área em metros quadrados.
* **[Uni3Exe02](Uni3Exe02.java) - Sistema de Desconto:** Calcula um desconto fixo de 12% sobre o valor de um produto e exibe o preço final.
* **[Uni3Exe03](Uni3Exe03.java) - Abastecimento de Combustível:** Determina quantos litros de gasolina podem ser abastecidos com base no valor pago e no preço do litro.
* **[Uni3Exe04](Uni3Exe04.java) - Média Ponderada:** Sistema de cálculo acadêmico que processa três notas com pesos diferentes (5, 3 e 2) para gerar a média final.
* **[Uni3Exe05](Uni3Exe05.java) - Custos de Produção (Granja):** Calcula o gasto total com anéis de identificação (chips e alimento) para um lote de frangos.
* **[Uni3Exe06](Uni3Exe06.java) - Precificação de Refeição:** Calcula o valor a ser pago em um restaurante a quilo, descontando automaticamente o peso do prato (tara de 0,750 kg).
* **[Uni3Exe07](Uni3Exe07.java) - Conversão de Volumes:** Lê a quantidade de garrafas e latas de diferentes tamanhos (350ml, 600ml e 2L) compradas por um comerciante e calcula o total em litros.

## 🚀 Como executar os projetos

1. Certifique-se de ter o [JDK (Java Development Kit)](https://www.oracle.com/java/technologies/downloads/) instalado na sua máquina.
2. Clone este repositório.
